## üé• Recording Tips

### Screen Recording
**Recommended Tools:**
- **Windows**: OBS Studio (free) or Camtasia
- **Terminal Recording**: Use Windows Terminal with a clean theme

### Terminal Setup
1. Use a clean, readable font (Cascadia Code, Fira Code)
2. Increase font size for visibility (16-18pt)
3. Use a warm color scheme matching the "Her" aesthetic:
   - Background: Cream (#FEF4E8)
   - Text: Warm brown (#8B4513)
   - Accent: Soft coral (#F8B195)

### Audio
- Record in a quiet environment
- Use a decent microphone if possible
- Speak clearly and naturally
- Add subtle background music (royalty-free, warm instrumental)

### Video Editing
**Recommended Software:**
- **Free**: DaVinci Resolve, OpenShot
- **Paid**: Adobe Premiere Pro, Final Cut Pro

**Editing Checklist:**
- [ ] Add title cards with warm color scheme
- [ ] Include soft transitions between sections
- [ ] Add text overlays for key points
- [ ] Include background music (low volume, 20-30%)
- [ ] Color grade to match "Her" aesthetic (warm, soft tones)
- [ ] Add subtle zoom-ins on important terminal output
- [ ] Export in 1080p or 4K

## üé® Visual Style Guide

### Color Palette (from "Her")
- Primary: Peachy Orange (#FFB5A7)
- Secondary: Soft Coral (#F8B195)
- Background: Cream (#FEF4E8)
- Accent: Muted Orange (#F67E7D)
- Text: Warm Brown (#8B4513)

### Typography
- Use clean, modern sans-serif fonts
- Suggested: Inter, Helvetica Neue, SF Pro

### Transitions
- Use soft fades and gentle transitions
- Avoid harsh cuts or flashy effects
- Keep it intimate and warm

## üì§ Publishing

### Video Hosting
- **YouTube**: Best for discoverability
- **GitHub**: Upload to repository (keep under 10MB or use Git LFS)
- **Vimeo**: Professional alternative

### Thumbnail
Create a custom thumbnail using:
- The hero banner as base
- Add "DEMO" or "WATCH NOW" text

- 1280x720 resolution

### File Naming
- `mia-2.0-demo.mp4` for the main video
- `demo-thumbnail.png` for the thumbnail
- Keep files under 50MB for GitHub (or use Git LFS)
